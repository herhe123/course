# 3.3线性表的抽象数据类型

```
ADT 线性表(List)
Data 
Operation
	InitList(*L)	初始化，建立空的线性表
	ListEmpty(L)	确认线性表是否空，true、false
	ClearList(*L)	清空
	GetElem(L,i,*e)	改变某位置的元素
	LocateElem(L,e)	查找元素，返回序号
	ListInsert(*L,i,e)	在某个位置插入新元素
	ListDelete(*L,i,*e)	删除某个位置的元素
	ListLength(L)	确定元素个数
endADT
```

例：A=AUB（在B中且不在A中的，插入到A中）

```
void union(List *La,List *Lb)
{
	int La_Len,Lb_Len,i;
	ElemType e;			声明数据元素e：既在A中，又在B中
	La_len = ListLength(List *La);
	Lb_len = ListLength(List *Lb);
	for(let i = 1; i <Lb_Len; i++){
		GetElem(Lb, i, e);	Lb的第i个元素赋值给e
		if(!LocateElem(La, e, equal)){	确保La中不存在和e相同的数据元素
			ListInsert(La, ++La_len, e);	将这个e插入到La中
		}
	}
}
```

# 3.4顺序存储结构

用一维数组来存储

```
#define MAXSIZE 20	设定存储空间容量
typedef int ElemType;	
typedef struct
{
	ElemType data{MAXSIZE}；	数组存储数据元素，最大值为MAXSIZE
	int length;	线性表当前长度
}SqList;
```

起始位置：data

最大存储容量：MaxSize

当前长度：length

| 数组长度                               | 线性表长度             |
| -------------------------------------- | ---------------------- |
| 存放线性表的存储空间长度               | 线性表中数据元素的个数 |
| 不变                                   | 可变                   |
| 线性表的第i个元素存储在数组的i-1的位置 |                        |

存储器中每个存储单元有自己的编号，称为地址

LOC(ai)=LOC(a1)+(i-1)*c

每个线性表位置的存入或取出数据的存储性能相同O(1)，称为随机存储结构
